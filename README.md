# MAZI — Static Mock Preview

Self-contained, single-file preview of the MAZI collectibles intelligence UI for visual / product review only.

## What this is

- Static HTML snapshot of the MAZI prototype interface.
- Uses **mock / sample data only** — every card, price, seller, and buyer is generated from a deterministic seed in the snapshot builder.
- No live data, no real users, no real sales.

## How to view

**GitHub Pages (if enabled):** open the published URL.

**Locally:** download `index.html` and double-click it. Any modern browser works.

## What you do not need

- No M4 access.
- No Tailscale, VPN, or SSH.
- No Whatnot account.
- No backend, server, or API key.
- No build step or `npm install`.

## What works in the preview

- Whatnot Live grid, source-specific tabs (eBay / Goldin / Fanatics / Heritage / All), VDEX tab, VDEX detail view with chart, and the sale-detail drawer.
- Moderation buttons render but are no-ops in the static demo.

## Data note

The bottom-right ribbon ("MAZI Snapshot · Mock Data") is a permanent reminder that nothing in this view reflects real sales.
